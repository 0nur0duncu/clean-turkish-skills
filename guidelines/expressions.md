# Atılacak Kelime ve Kalıplar

## Boğaz Temizleme Açılışları

Bunlar duyuru cümleleridir. İçeriği doğrudan söyleyin.

- "İşin aslı şu:"
- "Şöyle ki:" / "Şöyle:"
- "Şunu söylemekte fayda var:"
- "Şunu belirtmek gerekir ki"
- "Aslına bakarsanız"
- "Açık konuşalım" / "Açıkçası"
- "Şunu unutmamak gerekir"
- "Önemli bir nokta var:"
- "Şu bir gerçek ki"
- "İtiraf etmek gerekirse"
- "Hiç şüphesiz"
- "Hepimizin bildiği gibi"
- "Malum"

Herhangi bir "şunu söyleyeyim / şu var ki / şöyle ki" kuruluşu, noktadan önceki boğaz temizlemedir. Kesin ve noktayı söyleyin.

## Vurgu Koltuk Değnekleri

Anlam katmazlar. Silin.

- "Nokta." (cümle sonu vurgusu olarak)
- "Bu kadar." / "İşte bu kadar." / "Hepsi bu."
- "Bunu bir düşünün." / "Oturup düşünmek lazım."
- "Bu çok önemli çünkü"
- "Yanlış anlamayın"
- "Abartmıyorum"
- "İşte bu yüzden bu önemli"

## Zarflar

Boş pekiştireç ve çekinceleri öldürün. Bunlar vurgu katmaz, sadece şişirir.

Sık görülenler:

- "aslında"
- "gerçekten"
- "kesinlikle"
- "oldukça"
- "son derece"
- "adeta" / "âdeta"
- "resmen"
- "tam anlamıyla" / "kelimenin tam anlamıyla"
- "açıkçası"
- "esasen"
- "bir nevi" / "bir bakıma"
- "fazlasıyla" / "bir hayli" / "epeyce"
- "gerçek anlamda"
- "düpedüz"
- "tabiri caizse"
- "neredeyse" (dolgu olarak)

Şu dolgu kalıplarını da kesin:

- "Özünde" / "Temelde"
- "Günümüzde" / "Günümüz dünyasında" / "Şu çağda"
- "Belirtmek gerekir ki"
- "Günün sonunda" ("at the end of the day" kalkı)
- "Söz konusu olduğunda"
- "Genel olarak bakıldığında"
- "Nihayetinde"

## Meta-Yorum

Metnin kendi yapısını duyuran kenar notlarını silin. Metin yürümeli, kendini anlatmamalı.

- "Bu yazıda ... ele alacağız"
- "Bu bölümde ... inceleyeceğiz"
- "İlerleyen bölümlerde ..."
- "Birazdan göreceğimiz gibi"
- "Yukarıda da belirttiğimiz gibi"
- "Sizi adım adım götüreceğim"
- "Hadi başlayalım" / "Hadi dalalım"
- "İpucu:" / "Spoiler:" / "Sürpriz:"
- "Bunu zaten biliyorsunuz ama"
- "Ama o ayrı bir yazının konusu"

## Resmi Şişkinlik Ekleri

Türkçe AI metninin en güçlü belirtisi: bürokratik `-mektedir/-maktadır` eki ve beyanı şişiren `-dir` ek-fiili. Resmi-temiz üslupta "siz" kalır, bu ekler gider.

| Şişkin | Sade |
|--------|------|
| yapılmaktadır | yapılıyor / (özne) yapıyor |
| görülmektedir | görülüyor / (kim) görüyor |
| bilinmektedir | biliniyor / (kim) biliyor |
| edilmektedir | ediliyor |
| önem taşımaktadır | önemli |
| ifade edilebilir | diyebiliriz / (doğrudan söyle) |
| önemlidir | önemli |
| gereklidir | gerekli |
| mümkündür | mümkün |
| muhtemeldir | muhtemelen / belki |
| söz konusudur | var / (doğrudan söyle) |
| açıktır ki | (sil) |

Not: `-dir` her zaman yasak değildir; tanım ve genelgeçer olgularda meşrudur ("Su 100 derecede kaynar"). Yasak olan, sıradan beyanı resmîleştirmek için kullanmaktır ("Bu durum endişe vericidir" → "Bu endişe verici").

## Dolgu İsim ve Kalıplar

Anlam taşımayan, cümleyi uzatan ilgeç kalıpları. Silin veya somutlaştırın.

| Dolgu | Düzeltme |
|-------|----------|
| söz konusu | sil / "bu" |
| ... bağlamında | "... konusunda" / "... için" / (somutla) |
| ... noktasında | "... konusunda" / (yeniden kur: "karar verme noktasında" → "karar verirken") |
| ... açısından | (aşırı kullanımda somutla) |
| ... itibariyle | "... olarak" / sil |
| ... nezdinde | "... gözünde" / "... için" |
| ... anlamında | sil / somutla |
| ... çerçevesinde | "... kapsamında" / sil |
| ... doğrultusunda | "... göre" / sil |
| ... ekseninde | sil |
| ... özelinde | "... için" |
| ilgili (belirsiz) | (somutla: "ilgili birim" → adını söyle) |
| gerçekleştirmek (yapmak yerine) | "yapmak" |

## Belirsiz Beyanlar

Belirli şeyi adlandırmadan önem ilan eden cümleler. Kesin.

- "Nedenler yapısaldır"
- "Etkileri büyüktür"
- "Sonuçları önemlidir"
- "Bu en derin sorundur"
- "Riskler yüksektir"

Bir cümle bir şeyin önemli/derin/yapısal olduğunu söylüyor ama belirli şeyi göstermiyorsa, kesin veya o belirli şeyle değiştirin.
